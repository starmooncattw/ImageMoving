# ImageMoving - 圖片動畫工具集

兩款專業圖片動畫製作工具，滿足不同使用需求：

| 工具 | 用途 | 輸出格式 | 適合場景 |
|------|------|---------|---------|
| **Animation HTML Generator** | OBS 直播動畫 | HTML 檔案 | 直播、網頁 |
| **Physics GIF Studio** | GIF 動圖製作 | GIF 圖檔 | 社群分享 |

---

## 🎬 Animation HTML Generator | 動畫HTML產生器

為 OBS 直播設計的圖片動畫產生工具，輸出獨立 HTML 檔案，可直接在 OBS 中使用。

### ✨ 核心特色

- 🎯 **14種移動模式** - 彈跳、圓周、之字形、對角線等豐富路徑
- 🎨 **9種視覺特效** - 旋轉、縮放、扭曲、模糊、發光等效果可疊加
- 📦 **獨立HTML檔** - 一鍵產生，無需額外依賴
- 🌈 **透明背景** - 完美疊加在直播畫面
- 💾 **完全離線** - 無需網路，本地運行
- 🔧 **快速範本** - 桌面寵物、能量球、訂閱特效等預設

### 🚀 快速開始

#### 步驟 1：開啟產生器
用瀏覽器開啟 `AnimationHTML_Generator.html`

#### 步驟 2：上傳圖片
支援 **JPG / PNG / GIF / WebP**

#### 步驟 3：設定效果

**移動模式**（14種）
- 靜止、隨機出現、隨機彈跳
- 水平：左→右、右→左、來回
- 垂直：上→下、下→上、來回
- 對角線：4個方向
- 圓周運動（可調半徑）
- 之字形

**視覺特效**（可多選疊加）
- 旋轉：順時針/逆時針/擺動
- 縮放脈動：呼吸/彈跳效果
- 扭曲：水平/垂直拉伸
- 傾斜搖晃：左右/上下搖晃
- 模糊：高斯/動態模糊
- 陰影/發光：投影或外發光
- 色相旋轉：彩虹變色
- 3D旋轉：X/Y/Z軸翻轉
- 抖動：震動效果

**其他設定**
- 圖片大小：10% ~ 200%
- 透明度：0% ~ 100%
- 速度調整：1慢 ~ 10快
- 翻轉：水平/垂直/兩者

#### 步驟 4：產生檔案
點擊「產生動畫檔案」→ 下載 `.html` 檔案

#### 步驟 5：OBS 中使用
1. OBS → 新增來源 → **瀏覽器**
2. 勾選「本機檔案」
3. 選擇產生的 `.html` 檔案
4. 設定寬高（建議 1920x1080）
5. 完成！

### 📌 快速範本

一鍵套用預設效果：
- **桌面寵物** - 彈跳移動
- **訂閱特效** - 放大淡出
- **能量球** - 圓周發光
- **飄浮效果** - 垂直來回 + 淡入淡出
- **Padoru** - 隨機出現

### ❓ 常見問題

**Q: OBS 看不到動畫？**
A: 確認瀏覽器來源的寬高不是 0，且未勾選「關閉來源時關閉瀏覽器」

**Q: 動畫速度不對？**
A: 在產生器中調整速度滑桿後重新產生檔案

**Q: 可以用 GIF 動圖嗎？**
A: 可以！會保留 GIF 原本動態並加上設定效果

**Q: 檔案很大？**
A: 圖片會以 Base64 嵌入，建議壓縮至 < 1MB

## 🎨 Physics GIF Studio | 物理GIF工作室

專業 GIF 動圖錄製工具，內建物理引擎和粒子系統，適合製作社群分享的動態貼圖。

### ✨ 核心特色

- 🎯 **13種物理模式** - 日式物理引擎（蹦蹦跳、軟Q彈跳、果凍等）
- ✨ **8種粒子系統** - 下雪、愛心、櫻花、螢火蟲、彩紙等特效
- 🎬 **即時預覽** - 所見即所得，預覽與輸出同步
- 🖱️ **拖曳定位** - 滑鼠拖曳調整圖片位置
- 🎞️ **精細控制** - FPS、秒數、背景模式可調
- 🌈 **豐富濾鏡** - 模糊、波浪、故障、彩虹、遞迴等

### 🚀 快速開始

#### 步驟 1：開啟工作室
用瀏覽器開啟 `PhysicsGIF_Studio.html`

#### 步驟 2：上傳圖片
支援 **PNG / JPG**（⚠️ GIF 只會使用第一幀）

#### 步驟 3：選擇效果

**快速特效選單**（一鍵套用）

*物理效果*
- **蹦蹦跳** - 彈跳落地
- **軟Q彈跳** - 柔軟彈性
- **高彈簧** - 高彈跳伸縮
- **東張西望** - 左右搖頭
- **徘徊** - 徘徊晃動
- **輕拍** - 上下輕拍
- **自轉** - 水平翻轉
- **軟嫩果凍** - 果凍晃動
- **逼近** - 放大逼近
- **彈性** - 彈性變形
- **飄浮** - 上下飄浮
- **撞擊** - 震動撞擊
- **搖擺** - 鐘擺搖擺

*移動效果*
- **左滑入** / **右滑入** - 從邊緣滑入
- **鸚鵡軌跡** - 小圓周運動

*縮放/翻轉*
- **心跳** - 脈衝縮放
- **左右翻** - 連續翻轉

*視覺濾鏡*
- **彩虹** - 色相旋轉
- **故障** - 故障特效
- **老電影** - 懷舊風格
- **負片** - 負片閃爍
- **波浪** - 波浪扭曲
- **無限** - 遞迴效果

*粒子系統*
- **下雪** - 雪花飄落
- **愛心** - 愛心上升
- **泡泡** - 泡泡漂浮
- **櫻花** - 櫻花飄落
- **閃光** - 星星閃爍
- **螢火蟲** - 螢火蟲飛舞
- **彩紙** - 彩紙飛舞

**詳細控制台**

*圖片設定*
- **縮放** - 0.1 ~ 1.5 倍
- **拖曳定位** - 滑鼠直接拖曳

*物理模式*
- 13種物理動態
- 強度：0 ~ 100
- 速度：0.1 ~ 30（支援小數點）

*動作與幾何*
- **移動模式** - 無、左/右/上/下滑入、鸚鵡圓軌
- **移動距離** - 100 ~ 800
- **不倒翁搖擺** - 0 ~ 90 度
- **旋轉模式** - 無、順時針、逆時針
- **縮放/翻轉** - 無、心跳、放大、左右/上下翻轉

*視覺濾鏡*
- **模糊** - 0 ~ 20 px
- **馬賽克** - 1 ~ 30 倍
- **波浪** - 0 ~ 50 強度
- **雜訊** - 0 ~ 100
- **故障** - 0 ~ 50 偏移
- **彩虹** - 0 ~ 50 速度
- **無限遞迴** - 0 ~ 10 層

*粒子系統*
- 8種粒子效果
- **粒子速度** - 0 ~ 0.5（可調）

*背面與開關*
- **背面模式** - 鏡像/變暗/純色
- **鏡像** - 雙人對稱
- **倒影** - 水面倒影
- **負片/老電影/灰階** - 一鍵切換

#### 步驟 4：調整輸出

**畫布設定**
- 寬度/高度：自訂尺寸
- 秒數：動畫長度（秒）
- FPS：10 / 15 / 20 / 30

**背景模式**
- 透明背景
- 去背優化（白/黑）
- 純色背景（白/黑/綠幕）

#### 步驟 5：錄製下載
點擊「🔴 錄製並下載 GIF」→ 自動下載 `.gif` 檔案

### 💡 使用技巧

**組合效果**
- 物理模式 + 粒子系統 = 動態背景
- 移動 + 翻轉 = 翻滾動畫
- 濾鏡疊加 = 創意特效

**速度調整**
- 預覽比錄製快 → 降低速度值
- 粒子太快 → 調整粒子速度滑桿

**位置調整**
- 直接拖曳圖片到想要的位置
- 錄製時會保留位置

### 🎯 使用場景

- **Discord 貼圖** - 伺服器專屬動態表情
- **LINE 貼圖** - 個人貼圖製作
- **社群分享** - Twitter、IG 動態貼文
- **論壇簽名檔** - 個性化動態簽名
- **聊天表情包** - 自製動圖表情

### ⚠️ 注意事項

**GIF 限制**
- 上傳 GIF 只會使用第一幀（瀏覽器 Canvas 限制）
- 建議使用靜態圖片（PNG/JPG）

**錄製效能**
- 複雜效果會增加錄製時間
- 建議秒數 ≤ 3 秒，FPS ≤ 30
