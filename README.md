# 圖片動畫產生器

為 OBS 設計的圖片動畫效果產生工具，可建立各種動態圖片效果，適合直播、影片製作使用。

## 特色

- ✅ **完全離線使用** - 無需網路連線
- ✅ **簡單操作** - 視覺化介面設定
- ✅ **獨立檔案** - 產生的動畫檔案可獨立運行
- ✅ **豐富效果** - 14種移動模式 + 9種視覺特效
- ✅ **OBS 友善** - 透明背景，直接拖入使用

## 快速開始

### 1. 下載專案

下載 `config.html` 檔案到電腦

### 2. 開啟設定工具

用瀏覽器開啟 `config.html`

### 3. 設定動畫

#### 基礎設定
1. 點擊「圖片上傳」選擇圖片（支援 JPG/PNG/GIF/WebP）
2. 預覽區會顯示圖片

#### 移動設定
選擇移動模式：
- **靜止** - 圖片固定不動
- **隨機出現** - 在畫面隨機位置淡入淡出
- **彈跳移動** - 像彈力球一樣碰撞邊界反彈
- **水平移動** - 左→右、右→左、來回
- **垂直移動** - 上→下、下→上、來回
- **對角線移動** - 四個方向斜向移動
- **圓周運動** - 繞圈旋轉（可設定半徑）
- **之字形** - Z字路徑移動

調整速度（1慢 ~ 10快）

#### 圖片變換
- **翻轉** - 水平/垂直/兩者翻轉

#### 視覺特效（可多選疊加）
- **旋轉** - 順時針/逆時針/來回擺動
- **縮放脈動** - 呼吸/彈跳效果
- **扭曲** - 水平/垂直拉伸
- **傾斜搖晃** - 左右/上下搖晃
- **模糊** - 高斯/動態模糊
- **陰影/發光** - 投影或光暈效果
- **色相旋轉** - 彩虹色變化
- **3D旋轉** - X/Y/Z軸旋轉
- **抖動** - 震動效果

#### 其他設定
- **圖片大小** - 10% ~ 200%
- **透明度** - 0% ~ 100%
- **輸出檔名** - 自訂檔案名稱（預設 output）

### 4. 產生檔案

1. 點擊「產生動畫檔案」
2. 瀏覽器會下載 `.html` 檔案

### 5. 在 OBS 中使用

1. 開啟 OBS
2. 新增來源 → **瀏覽器**
3. 勾選「本機檔案」
4. 點擊「瀏覽」選擇產生的 `.html` 檔案
5. 設定寬度/高度（建議 1920x1080）
6. 完成！

## 快速範本

點擊「快速範本」按鈕快速套用預設效果：

- **桌面寵物** - 彈跳移動效果
- **飄浮效果** - 垂直來回 + 淡入淡出
- **霓虹燈** - 色相旋轉 + 發光效果
- **翻滾動畫** - 水平移動 + 旋轉
- **隨機出現** - 隨機位置淡入淡出

## 進階使用

### 重複使用設定

如果想要修改已產生的動畫：
1. 在 `config.html` 中重新設定
2. 用相同檔名產生新檔案覆蓋即可

### 多個動畫

可以產生多個不同設定的檔案：
1. 設定第一個動畫 → 產生 `動畫1.html`
2. 重置設定
3. 設定第二個動畫 → 產生 `動畫2.html`
4. 在 OBS 中可同時使用多個動畫檔案

### 調整動畫在 OBS 中的位置

在 OBS 的來源清單中：
- 拖曳調整位置
- 拉伸調整大小
- 右鍵 → 變換 → 可旋轉、翻轉

## 常見問題

### Q: 為什麼產生的檔案在瀏覽器開啟正常，但 OBS 看不到？

A: 確認 OBS 瀏覽器來源的設定：
- 寬度/高度不能是 0
- 「關閉來源時關閉瀏覽器」不要勾選（可能導致重新載入）

### Q: 動畫速度太快/太慢？

A: 在 `config.html` 中調整速度滑桿後重新產生檔案

### Q: 可以用 GIF 動圖嗎？

A: 可以！上傳 GIF 後會保留原本的動態效果，再加上設定的移動/特效

### Q: 檔案很大怎麼辦？

A: 圖片會以 Base64 編碼嵌入 HTML，建議：
- 壓縮圖片（< 1MB 為佳）
- 降低圖片解析度
- 使用 PNG/WebP 格式

### Q: 在 config.html 設定無法顯示？

A: 這是正常的！`config.html` 只是設定工具，實際動畫在**產生的檔案**中。
